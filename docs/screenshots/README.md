# Captures d’écran

Les fichiers `hero.png`, `vue-3d.png` et `projets.png` sont des **aperçus générés** pour le README. Remplacez-les par de vraies captures une fois l’interface stabilisée :

1. Lancez l’environnement : `./scripts/run.sh`
2. Ouvrez http://localhost:9000
3. Capturez les vues pertinentes (même ratio ou recadrez)
4. Écrasez les PNG dans ce dossier en conservant les noms pour éviter de modifier le README

Conseil : export PNG ou WebP, largeur ~1200–1600 px pour une lecture nette sur GitHub.
